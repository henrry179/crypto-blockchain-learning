# Crypto & Blockchain Learning Path

一个系统化、结构化的区块链与加密货币学习资源库，涵盖从支付系统基础到前沿区块链技术的完整知识体系。

## 🚀 项目概述

本项目旨在为区块链和加密货币学习者提供一个全面、有组织的学习路径和资源集合。资源按照类型和主题精心分类，方便持续学习和跟踪这个快速发展的领域。

**项目状态：** ✅ 已完成基础搭建和内容填充
**最后更新：** 2025-09-05 17:30:08

## 🧠 系统提示词与开发任务清单

### 系统提示词通用模板

用于指导AI助手在处理项目开发任务时的行为规范，特别是涉及更新开发时间进度表的需求。

核心原则：
1. 探索项目当前开发进度
2. 制定开发规划
3. 创建TODOS任务清单
4. 执行开发任务
5. 每次完成一个任务进行代码审查
6. 同步项目主文档并更新开发进度时间表
7. 循环迭代直到完成所有任务
8. 最后推送到远程仓库

特殊规则：
- 更新开发时间进度表时使用本机电脑当前的实时时间
- 暂停对npm、node.js等复杂环境配置工具的开发
- 优先开发可直接实现的功能或模块
