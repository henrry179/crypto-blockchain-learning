# Crypto & Blockchain Learning Path

一个系统化、结构化的区块链与加密货币学习资源库，涵盖从支付系统基础到前沿区块链技术的完整知识体系。

## 🚀 项目概述

本项目旨在为区块链和加密货币学习者提供一个全面、有组织的学习路径和资源集合。资源按照类型和主题精心分类，方便持续学习和跟踪这个快速发展的领域。

**项目状态：** ✅ 已完成基础搭建和内容填充
**最后更新：** 2025-09-05 15:24:03

### II. 技术开发与进展追踪
- 核心协议更新追踪

### IV. 行业动态与观点

- 现代支付网络分析

## 🛠 使用方法

1. **克隆本仓库**
   ```bash
   git clone https://github.com/henrry179/crypto-blockchain-learning.git
   ```

2. **按照数字序号系统学习**
   - 从01-foundation开始建立基础
   - 逐步深入到各专业领域
